# Proyecto Tienda Java - Warhammer

Este proyecto es un **e-commerce de figuras de Warhammer**, desarrollado en **Java 21** con **Spring Boot** y utilizando una base de datos **H2** en memoria. Para la generación de vistas se combinan plantillas de **Mustache** y **Thymeleaf**, y el **JavaScript** se apoya en **jQuery** para manejar operaciones tanto del lado del cliente como del servidor.

## Tecnologías y versiones utilizadas

- **Java:** 21  
- **Spring Boot:** 3.5.6  
- **Plantillas:** Mustache + Thymeleaf  
- **Base de datos:** H2  
- **JavaScript:** jQuery  
- **Entorno de desarrollo:** Eclipse IDE  

## Resumen del proyecto

La aplicación permite:

- Navegar y visualizar figuras de Warhammer disponibles.  
- Gestionar el carrito de compras y procesar pedidos.  
- Mostrar información dinámica usando plantillas HTML.  
- Mantener los datos de productos y usuarios en una base de datos H2.

Es un proyecto de ejemplo para demostrar la integración de **Java**, **Spring Boot**, **plantillas dinámicas** y **JavaScript** en un entorno de desarrollo profesional.


## Requisitos para ejecutar

1. Tener instalado **Java 21** y **Eclipse IDE**.  
2. Clonar o descargar el proyecto.  
3. Abrir el proyecto en **Eclipse** y ejecutar la aplicación Spring Boot.  
4. Acceder a la aplicación en `http://localhost:8080`.
